ICMP 类型

发表于 2018-08-13 更新于 2020-01-22 分类于 Linux Disqus：

Address Unreachable 地址无法访问

This error message is generated by a router, or by the IPv6 layer in the originating node, in response to a packet that cannot be delivered to its destination address for reasons other than congestion.
此错误消息由路由器或始发节点中的IPv6层生成，以响应由于拥塞以外的原因而无法传送到其目标地址的数据包。
Bad Header 坏标题
This error message is created if there has been an error in the header of a packet
如果数据包的标头中存在错误，则会创建此错误消息
Beyond Scope 超越范围
This error message is sent if transmitting a package whould cross a zone boundary of the scope of the source address.
如果传输包跨越源地址范围的区域边界，则会发送此错误消息。
Communication Prohibited 禁止通信
This error message is sent if communication with destination administratively prohibited.
如果在管理上禁止与目标通信，则会发送此错误消息
Destination Unreachable 目的地无法到达
This error message is generated by a host or gateway if the destination is not reachable.
如果无法访问目标，则由主机或网关生成此错误消息
Echo Reply (pong) Echo回复（pong）
This message is the answer to an Echo Request.
此消息是Echo请求的答案。
Echo Request (ping) 回应请求（ping）
This message is used to test if a host is reachable mostly with the ping utility.
此消息用于测试主机是否可通过ping实用程序访问。

Failed Policy 策略失败
This error message is generated if the source address failed ingress/egress policy.
如果源地址入口/出口策略失败，则会生成此错误消息。
Fragmentation Needed 需要分段
This error message is sent if fragmentation is required, and Don not Fragment (DF) flag is set.
如果需要分段，则发送此错误消息，并设置Don not Fragment（DF）标志。
Host Precedence Violation 主机优先权违规
This error message is sent if the communication administratively prohibited.
如果管理上禁止通信，则会发送此错误消息。
Host Prohibited 主机禁止
This error message is sent if access from a host administratively prohibited.
如果在管理上禁止从主机访问，则会发送此错误消息。
Host Redirect 主机重定向
This message is sent if the datagram is redirected for the host.
如果为主机重定向数据报，则会发送此消息。
Host Unknown 主机未知
This error message is sent if the destination host is unknown.
如果目标主机未知，则会发送此错误消息。
Host Unreachable 主机无法访问
This error message is sent if the destination host is unreachable.
如果目标主机无法访问，则会发送此错误消息。
Ip Header Bad
This error message is sent if the IP header is bad.
如果IP标头错误，则会发送此错误消息。
Neighbour Advertisement (Neighbor Advertisement) 邻居广告（邻居广告）
This informational message is sent in response to a neighbour-solicitation message in order to (unreliably) propagate new information quickly.
响应于邻居请求消息发送该信息性消息，以便（不可靠地）快速传播新信息。
Neighbour Solicitation (Neighbor Solicitation) 邻居请求（邻居请求）
This informational message is sent by a node to determine the link-layer address of a neighbor, or to verify that a neighbor is still reachable via a cached link-layer address. Neighbor Solicitations are also used for Duplicate Address Detection.
该信息性消息由节点发送以确定邻居的链路层地址，或者验证通过高速缓存的链路层地址仍可到达邻居。 Neighbor Solicitations也用于重复地址检测。
Network Prohibited 网络禁止
This message is sent if the network is administratively prohibited.
如果管理上禁止网络，则会发送此消息。
Network Redirect 网络重定向
This message is sent if the datagram is redirected for the network.
如果为网络重定向数据报，则会发送此消息。
Network Unknown 网络未知
This message is sent if the destination network is unknown.
如果目标网络未知，则发送此消息。
Network Unreachable 网络无法访问
This message is sent if the destination network is unreachable.
如果目标网络无法访问，则会发送此消息。
No Route 没有路由
This error message is set if there is no route to the destination.
如果没有到目标的路由，则会设置此错误消息。
Packet Too Big 数据包太大了
This error message is sent by a router in response to a packet that it cannot forward because the packet is larger than the MTU of the outgoing link.
此错误消息由路由器发送，以响应无法转发的数据包，因为数据包大于传出链路的MTU。
Parameter Problem 参数问题
This error message is generated if the IP header is bad, either by a missing option or bad length.
如果IP标头错误，则会生成此错误消息，无论是缺少选项还是错误长度。
Port Unreachable 端口无法访问
This error message is sent if the port unreachable.
如果端口无法访问，则会发送此错误消息。
Precedence Cutoff 优先截止
This message is sent if the precedence is lower than the required minimum.
如果优先级低于所需的最小值，则发送此消息。
Protocol Unreachable 协议无法访问
This message is sent if the destination protocol is unreachable.
如果目标协议无法访问，则会发送此消息。
Redirect 重定向
This error message informs a host to send packets on another route.
此错误消息通知主机在另一个路由上发送数据包。
Reject Route 拒绝路由
This error message is sent if the route to destination is rejected.
如果拒绝到目的地的路由，则会发送此错误消息。
Required Option Missing 必需选项缺失
This message is sent if a required option is missing.
如果缺少必需选项，则会发送此消息。
Router Advertisement 路由器广告
This message is used by routers to periodically announce the IP address of a multicast interface.
路由器使用此消息定期通告多播接口的IP地址。
Router Solicitation 路由器征集
This message is used by a host attached to a multicast link to request a Router Advertisement.
连接到多播链路的主机使用此消息来请求路由器通告。
Source Quench 来源Quench
This error message is generated to tell a host to reduce the pace at which it is sending packets.
生成此错误消息是为了告诉主机减少发送数据包的速度。
Source Route Failed 源路由失败
This message is sent if the source route has failed.
如果源路由失败，则发送此消息。
Time Exceeded 超过时间
This error message is generated if the time-to-live was exceeded either of a packet or of the reassembling of a fragmented packet.
如果超过数据包或重新组装分段数据包的生存时间，则会生成此错误消息。
Timestamp Reply 时间戳回复
This message is used to reply to a timestamp message.
此消息用于回复时间戳消息。
Timestamp Request时间戳请求
This message is used for time synchronization.
此消息用于时间同步。
TOS Host Redirect TOS主机重定向
This message is the datagram is redirected for the type of service and host.
此消息是数据报被重定向为服务类型和主机。
TOS Host Unreachable TOS主机无法访问
This message is sent if the host is unreachable for the type of service.
如果主机无法访问该服务类型，则会发送此消息。
TOS Network RedirectTOS网络重定向
This message is sent if the datagram is redirected for the type of service and network.
如果针对服务类型和网络重定向数据报，则会发送此消息。
TOS Network Unreachable TOS网络无法访问
This error message is sent if the network is unreachable for the type of service.
如果网络无法访问该类型的服务，则会发送此错误消息。
TTL Zero During Reassembly 重组期间TTL为零
This error message is sent if a host fails to reassemble a fragmented datagram within its time limit.
如果主机无法在其时间限制内重新组合碎片数据报，则会发送此错误消息。
TTL Zero During Transit 在过境期间TTL零
This error message is sent if the time to live exceeded in transit.
如果在传输过程中超时，则会发送此错误消息。
Unknown Header Type 未知标题类型
This error message is sent if an unrecognized Next Header type encountered.
如果遇到无法识别的Next Header类型，则会发送此错误消息。
Unknown Option 未知选项
This error message is sent if an unrecognized IPv6 option encountered.
如果遇到无法识别的IPv6选项，则会发送此错误消息。
